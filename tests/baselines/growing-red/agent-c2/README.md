# taskflow

Small task-queue workspace: an HTTP api, a background worker, and a shared library.

New here? Start with the end-to-end walkthrough: [docs/life-of-a-task.md](docs/life-of-a-task.md).

## Requirements

- Node >= 20.6.0
- `DATABASE_URL` set in the environment

## Setup

```sh
make setup
make dev
```

## Development

```sh
make test
make lint
```
